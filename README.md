# Cicada USB Boot Tool

**Мастер загрузочных USB-носителей Cicada3301** — для Windows и Android.

Программа автоматически размечает флешку, устанавливает загрузчик GRUB4EFI, распаковывает файлы Cicada3301 и позволяет добавлять образы Windows, Linux и WinPE через удобный графический интерфейс.

| Платформа | Версия UI | Файл запуска |
|-----------|-----------|--------------|
| **Windows** | Frosted Glass v2.1 PRO | `cicada_usb_tool_frosted.py` |
| **Android** | Compose UI | APK из `app/` |

## Возможности

### Создание загрузочной флешки
- Автоматический поиск USB-дисков
- Разметка **MBR**: NTFS (данные) + FAT32 (загрузчик)
- Загрузка архивов `Cicada3301.7z`, `FAT32.7z` и `7z.exe` с GitHub
- Распаковка через 7-Zip
- Скрытие раздела **BOOT** (FAT32) после установки

### Управление образами
- Добавление образов **Windows** (XP, Vista, 7, 8, 10, 11)
- Добавление образов **Linux** (Debian, Kali, Ubuntu)
- Добавление образов **WinPE**
- Удаление образов с флешки
- Прогресс-бар при копировании больших ISO

### Интерфейс (Windows)
- Тёмная тема **Premium Dark Cyber** с эффектом frosted glass
- Боковое меню навигации
- Счётчики образов и свободного места
- 7 этапов создания флешки с визуальным прогрессом

## Схема разделов

После создания флешки на диске появляются два раздела:

```
┌─────────────────────────────────────────────────────────────┐
│  USB-диск (MBR)                                             │
├──────────────────────────────┬──────────────────────────────┤
│  Раздел 1 — NTFS             │  Раздел 2 — FAT32 (скрытый)  │
│  Метка: Cicada3301           │  Метка: BOOT                 │
│  ~весь диск − 1536 МБ        │  1536 МБ                     │
│  Видимый, с буквой диска     │  Скрыт после установки       │
│  Папки: WINDOWS, LINUX,      │  GRUB4EFI, загрузочные       │
│  WINPE, e2b, ...             │  файлы из FAT32.7z           │
└──────────────────────────────┴──────────────────────────────┘
```

| Параметр | Значение |
|----------|----------|
| Таблица разделов | MBR |
| Раздел данных | NTFS, метка `Cicada3301` |
| Загрузочный раздел | FAT32, метка `BOOT`, 1536 МБ |
| Минимальный размер флешки | ~2 ГБ |

## Windows — установка и запуск

### Требования

| Компонент | Версия |
|-----------|--------|
| ОС | Windows 10 / 11 |
| Python | 3.10+ |
| PyQt6 | 6.6.0+ |
| Права | **Администратор** (обязательно) |
| Интернет | Нужен при первом создании флешки (загрузка архивов) |

### Установка зависимостей

```powershell
cd "путь\к\apk"
pip install -r ..\..\requirements.txt
```

или напрямую:

```powershell
pip install "PyQt6>=6.6.0"
```

### Запуск

```powershell
cd "путь\к\apk"
python cicada_usb_tool_frosted.py
```

Без консольного окна (рекомендуется):

```powershell
pythonw cicada_usb_tool_frosted.py
```

> При запуске без прав администратора программа предложит перезапуск с повышением прав — это необходимо для разметки диска и настройки ACL.

### Опционально: иконка

Положите в папку с программой один из файлов:
- `icon.ico`
- `cicada_icon.ico`
- `cicada_icon.png`

## Windows — использование

### 1. Создание загрузочной флешки

1. Подключите USB-флешку (все данные на ней будут **удалены**).
2. Нажмите **Обновить** — выберите диск в списке.
3. Нажмите **СОЗДАТЬ ЗАГРУЗОЧНУЮ ФЛЕШКУ**.
4. Подтвердите форматирование.
5. Дождитесь завершения всех 7 этапов:
   - Поиск USB
   - Создание разделов
   - Форматирование
   - Установка загрузчика
   - Копирование файлов
   - Создание меню
   - Завершение
6. Безопасно извлеките флешку.

### 2. Добавление образов

| Меню | Куда копируется | Варианты |
|------|-----------------|----------|
| **WINDOWS** | `Cicada3301\WINDOWS\<версия>\` | XP, Vista, 7, 8, 10, 11 |
| **LINUX** | `Cicada3301\LINUX\<дистрибутив>\` | Debian, Kali, Ubuntu |
| **WINPE** | `Cicada3301\WINPE\` | — |

1. Выберите пункт в боковом меню.
2. Укажите версию / дистрибутив (для Windows и Linux).
3. Выберите файл образа на компьютере.
4. Дождитесь окончания копирования.

### 3. Удаление образа

1. Меню **УДАЛИТЬ ОБРАЗ**.
2. Выберите файл на флешке.
3. Подтвердите удаление.

## Android — сборка APK

Android-приложение создаёт загрузочную USB через **OTG без root** (подход как у [Ventoid](https://github.com/GPLaider/Ventoid): USB Host API + запись секторов).

### Требования

- Android 8.0+ (API 26)
- USB OTG
- Флешка ≥ 2 ГБ
- JDK 17+, Android SDK

### Отличия от Windows-версии

| Windows | Android |
|---------|---------|
| NTFS, раздел 1 | **exFAT** (ISO > 4 ГБ) |
| PowerShell / diskpart | SCSI WRITE через OTG |
| Права администратора | Разрешение USB |

### Сборка

```powershell
cd "путь\к\apk"
copy local.properties.example local.properties
# укажите sdk.dir в local.properties

.\gradlew.bat assembleDebug
```

APK: `app\build\outputs\apk\debug\app-debug.apk`

### Использование на Android

1. Подключите флешку через OTG.
2. Разрешите доступ к USB.
3. Выберите диск → **Создать флешку**.
4. Дождитесь загрузки архивов и установки.

## Устранение неполадок

### «Требуются права администратора»
Запустите программу от имени администратора или согласитесь на перезапуск с UAC.

### Диск не отображается в списке
- Убедитесь, что флешка подключена и определяется в «Управление дисками».
- Нажмите **Обновить**.
- Отключите другие программы, которые могут держать диск занятым.

### Ошибка разметки
Программа сначала пробует **PowerShell**, при неудаче — **diskpart**. Закройте проводник и другие окна, открытые на этой флешке.

### Ошибка при загрузке образа
- Проверьте, что на разделе достаточно свободного места.
- Убедитесь, что файл с таким именем ещё не существует на флешке.
- Запускайте программу с правами администратора.

### `name 'background' is not defined` (PyQt6)
Ошибка в CSS f-string — в `setStyleSheet(f"""...""")` фигурные скобки CSS должны быть удвоены: `{{` и `}}`. Исправлено в `IsoLoadDialog`.
